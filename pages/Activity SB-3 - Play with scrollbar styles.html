The Scrollbar tab

Styling the scrollbars

SB-3
Play with some more scrollbar styles

Let’s see what other scrollbar styles we can override...

  1. In the Scrollbar tab, make the gutters visible by setting their background color to something other than transparent. Add the following the the previous Activity’s thumb rule: 
    /* scrollbar style */
div.finbar-vertical, div.finbar-horizontal {
    /* gutter style when mouse somewhere else */
    background-color: white;
    opacity: 0;
    transition: opacity .5s;
}
div.finbar-vertical:hover, div.finbar-horizontal:hover {
    /* gutter style when mouse hovered over scrollbar */
    opacity: .5;
    transition: opacity .5s;
}
    We make the gutters translucent so the grid content can still be seen through them.
  2. Click the injectCSS('custom', …) button to update the custom style element
  3. Scrollbars now have visible gutters when hovered over.

There are many other possibilities. For some more examples, go back to the first page of this section and take a look at its Code Snippets tab.
