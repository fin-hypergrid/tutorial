The Data tab

D-2
Edit a value in the Data editor and watch the grid update

Remove the comma from the grid cell:

  1. Click the Data tab.
  2. Edit the name property in the first (dataRow) object.
  3. Click the grid.setData(…) button.
  4. The grid cell updates.
In actual practice, the data object you pass to setData will typically be JSON.parse’d from a character string in JSON format.

The Data tab uses eval to parse their editors as straight JavaScript, rather than JSON.parse which requires strict JSON. This is intended to make editing easier as you do not need to concern yourself with strict adherence to the JSON standard (double quotes, primitive types, no comments, _etc._).
