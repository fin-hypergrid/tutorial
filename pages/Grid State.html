Grid State

The State tab provides an editor for the grid.properties object, that is, the grid’s “own” properties that override the theme properties, which in turn override the default properties. For example: 
{
    color: 'red'
}
This sets the grid’s text color to red.

A grid state object may also include column properties in its columns property. These are not set on grid.properties but rather on the appropritate column.properties object where column references a specific column object. For example: 

{
  columns
}
